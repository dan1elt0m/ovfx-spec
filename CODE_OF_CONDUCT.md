# Code of Conduct

OVFX adopts version 2.1 of the **Contributor Covenant** as its code of conduct.

The full text is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — opening issues, sending pull requests, commenting in discussions — you agree to abide by it.

## Reporting

Please report unacceptable behaviour to the maintainers via the contact method listed in [`SECURITY.md`](SECURITY.md). Reports are handled confidentially.
