# OVFX repository skills

These skills are scoped to the OVFX repository. They are auto-discovered by Claude Code when it runs in this working directory.

| Skill                                    | Invoke with     | What it does                                                                  |
| ---------------------------------------- | --------------- | ----------------------------------------------------------------------------- |
| [`update-ovfx/`](update-ovfx/SKILL.md)   | `/update-ovfx`  | Bump the OVFX standard to a new PATCH / MINOR / MAJOR version, end to end.    |

Repository skills are a convenience. The canonical procedure they implement is in [`../../CLAUDE.md`](../../CLAUDE.md) — read that first if you want to understand the "why" behind any step.
